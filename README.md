**Семантическкое версионирование**
```textmate
X — мажорная версия (major): увеличивается при значительных изменениях, которые несовместимы с предыдущими версиями (например, изменение API, которое ломает существующий код).
Y — минорная версия (minor): увеличивается при добавлении новых функций или улучшений, совместимых с предыдущими версиями.
Z — патч (patch): увеличивается при исправлении ошибок или мелких изменениях, которые не влияют на функциональность и совместимость.
```
